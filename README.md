# Noon - E-commerce Web API

A powerful, enterprise-grade backend for an E-commerce marketplace, designed for high traffic and secure transactions.

## 🚀 Key Modules
- **Account Module:** User registration, login, and profile management via ASP.NET Identity.
- **Product Module:** Advanced filtering, sorting, and pagination for large catalogs.
- **Basket Module:** High-speed shopping cart management powered by **Redis**.
- **Order Module:** Workflow for order creation, tracking, and history.
- **Payment Module:** Live payment processing integration with **Stripe**.

## 🛠️ Tech Stack
- **Framework:** ASP.NET Core Web API
- **Database:** SQL Server & Entity Framework Core
- **Caching:** Redis
- **Security:** JWT (JSON Web Tokens)
- **Documentation:** Swagger UI (OpenAPI)

## 🏗️ Architecture Patterns
- Clean Architecture (Onion Architecture)
- Repository & Unit of Work Patterns
- Dependency Injection
- DTOs (Data Transfer Objects) for secure data exposure
